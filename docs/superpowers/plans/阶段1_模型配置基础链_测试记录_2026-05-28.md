1. 调用 `POST /api/model/providers` 新增 `Stage1TestProvider`
2. 调用 `GET /api/model/options` 校验 provider 已出现在 `providers`
3. 调用 `POST /api/model/select` 将当前模型切到 `Stage1TestProvider / stage1-model-b`
4. 调用 `GET /api/model/options` 校验 `current_llm_info` 已保存
5. 调用 `DELETE /api/model/providers/Stage1TestProvider`
6. 再次调用 `GET /api/model/options` 校验：
   - `providers` 中不再包含 `Stage1TestProvider`
   - `current_llm_info` 已清空，不再残留已删除 provider
