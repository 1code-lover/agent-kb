## 测试步骤摘要

### 1. 切换到模型 A

步骤：

1. 新增 `Stage4MockProvider`
2. 调用 `POST /api/model/select` 选择 `stage4-model-a`
3. 调用 `GET /api/model/options`
4. 调用 `POST /api/agent/run`，`mode=agent`

结果：

- `current_llm_info.model = stage4-model-a`
- `run_a.answer = MOCK_OK::stage4-model-a::confirm active model a`
- `run_a.steps[0].summary = Stage4MockProvider / stage4-model-a`
- `run_a.receipts[-1].output.model = stage4-model-a`

### 2. 切换到模型 B

步骤：

1. 再次调用 `POST /api/model/select` 选择 `stage4-model-b`
2. 调用 `GET /api/model/options`
3. 再次调用 `POST /api/agent/run`，`mode=agent`

结果：

- `current_llm_info.model = stage4-model-b`
- `run_b.answer = MOCK_OK::stage4-model-b::confirm active model b`
- `run_b.steps[0].summary = Stage4MockProvider / stage4-model-b`
- `run_b.receipts[-1].output.model = stage4-model-b`
